Willkommen zu Mathsite!

Diese Webseite soll dazu dienen faulen Leuten mit Mathematikformeln zu helfen. Es soll das Benutzen von solchen Formeln vereinfachen. Der Benutzer kann auf der Startseite eine Formel wählen. Die Formel wird angezeigt und darunter kann der Benutzer die Variablen eingeben. Das Resultat wird dann auf Knopfdruck generiert.

Zum verwenden einer Formel, wählen sie bitte Links eine Formel aus. Geben sie dann die benötigten Variablen an und klicken sie für ein Resultat.
